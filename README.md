Projeto – Análise com NotebookLM
Sobre
Este repositório contém o trabalho desenvolvido utilizando o NotebookLM como ferramenta de organização, análise de fontes e aplicação de engenharia de prompts.
O projeto teve como objetivo explorar o tema da inflação a partir de fontes institucionais e acadêmicas, validando o uso do modelo por meio de diferentes variações de prompts e análise crítica das respostas geradas.
